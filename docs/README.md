# Documentation Guide

This directory contains the durable product and planning source of truth for Verion.

## Read Order

Before a coding session, read the root operating files first:

1. [`EXECUTION.md`](../EXECUTION.md) — current milestone, progress, blockers, and next steps.
2. [`SPRINT.md`](../SPRINT.md) — the work required to ship now.
3. [`HACKATHON.md`](../HACKATHON.md) — historical submission reference; it no longer drives product decisions.

Then use the documents below for the decision at hand.

## Product Source of Truth

- [`PRODUCT_VISION.md`](product/PRODUCT_VISION.md) — full long-term product vision and experience.
- [`VISION.md`](product/VISION.md) — concise mission, personality, and product boundaries.
- [`MVP.md`](product/MVP.md) — non-negotiable Build Week loop and success criteria.
- [`PRODUCT_PRINCIPLES.md`](product/PRODUCT_PRINCIPLES.md) — decision-making principles.
- [`USER_STORY.md`](product/USER_STORY.md) — the intended developer experience.

## Planning and Governance

- [`DEMO.md`](planning/DEMO.md) — three-minute product demo narrative.
- [`DEMO_RUNBOOK.md`](planning/DEMO_RUNBOOK.md) — exact setup, repair, rerun, and reset procedure for rehearsal.
- [`PHASE_1_BUILD_PLAN.md`](planning/PHASE_1_BUILD_PLAN.md) — proposed vertical slice and approval gate for current implementation.
- [`PRODUCT_FOUNDATION_PLAN.md`](planning/PRODUCT_FOUNDATION_PLAN.md) — target-agnostic product implementation order and constraints.
- [`EVIDENCE_ARCHITECTURE.md`](planning/EVIDENCE_ARCHITECTURE.md) — hard integration boundary for producers, orchestration, Context Capsules, and GPT.
- [`SECURITY_INTEGRATION_PLAN.md`](planning/SECURITY_INTEGRATION_PLAN.md) — product boundary and safeguards for Verion's built-in security review.
- [`ROADMAP.md`](planning/ROADMAP.md) — phased direction beyond the current milestone.
- [`DECISIONS.md`](planning/DECISIONS.md) — major product and engineering decisions.
- [`QUESTIONS.md`](planning/QUESTIONS.md) — unresolved material choices awaiting approval.
- [`NOT_BUILDING.md`](planning/NOT_BUILDING.md) — deferred and prohibited scope.
- [`IDEAS.md`](planning/IDEAS.md) — interesting ideas that are not approved MVP work.
